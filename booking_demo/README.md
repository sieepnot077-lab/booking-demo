# Booking Demo

เดโมระบบจองคิวแบบง่ายด้วย Python Flask + SQLite

## ความสามารถ

- ลูกค้าจองคิวผ่านฟอร์ม
- ระบบบันทึกข้อมูลลง SQLite
- แอดมินดูรายการคิวทั้งหมด
- แอดมินกรองตามสถานะหรือวันที่ได้
- แอดมินเปลี่ยนสถานะคิวได้

## โครงสร้างไฟล์

- `app.py` แอป Flask และ routes
- `database.py` ฟังก์ชันเชื่อม SQLite
- `templates/` หน้า HTML
- `static/styles.css` สไตล์หน้าเว็บ
- `booking.db` จะถูกสร้างอัตโนมัติในโฟลเดอร์ชั่วคราวของระบบเพื่อเลี่ยงปัญหาสิทธิ์เขียนไฟล์

หมายเหตุ:
ถ้าสภาพแวดล้อมไม่อนุญาตให้ SQLite เขียนไฟล์ ระบบจะ fallback ไปใช้ SQLite แบบ in-memory ชั่วคราวเพื่อให้เดโมยังรันได้

## วิธีรัน

```bash
cd booking_demo
python app.py
```

แล้วเปิดเบราว์เซอร์ที่:

`http://127.0.0.1:5000`
